Intro to CSS

CSS Locations

CSS can be written in 3 locations.

Each location has a weight that decides when a conflict happens what style should display.

  1. External File - CSS can be written in a file that is included in the HTML page. (LINK tag)
  2. Internal File - Written within the HEAD tag this block of code defines an ares with the HTML page where we can write CSS. (STYLE tag)
  3. Inline Style - An attribute is added to an element where the CSS can be applied to the element directly. (STYLE attribute)

CSS Selectors

The way we connect the HTML and CSS is with a selector.

There are three basic types of selectors we use.

  • Tag Name Selector - the HTML tag is used to connect the HTML and the CSS. Most tags are valid to use. (h1, h2, p, etc)
  • Class Name Selector - An attribute of class is added to an HTML element with a name. That name is used to connect the CSS. (class="className", .className{})
  • ID Name Selector - An attribute of id is added to an HTML element with a name. That name is used to connect the CSS. (id="idName", #idName{})

Note: - All class and id are case sensitive.
