Types of Javascript Data

  • string
  • number
  • boolean
  • array
  • object

string

a string is collection of characters like 'ABC' or 'R2D2'

var mySting = "This is a string";

number

a number is number that math can be done on.

var myNumber = 129;

boolean

a boolean is a true or false value

var myBoolean = true;

array

a array is ordered collection of js data types

arrys can contain strings, numbers, booleans, arrays or objects

var myArray = ['string', 2, false, [1,2], {data: 'yes'}];

object

an object is a collection of data

var myObj = {name: 'Dan'};

myObj.name // returns Dan

If statements

javascript can look at a variable and act on it base on if it is true or false

function
